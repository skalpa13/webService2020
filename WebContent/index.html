A Simple RESTful example

This webapp is a simple RESTful application example.
It's hosted by Heroku and was designed with Java EE Technologies (Servlet, JPA, JAX-RS, JAX-B, ...).

The documentation of the REST API is available in the side bar to the right.

To consume web services, you can use cURL, develop your own REST client, or use one available as plugin for your browser. The best known are:

  • Simple REST Client for Google Chrome
  • RESTClient for Mozilla Firefox
